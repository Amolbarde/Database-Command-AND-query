# postgreSQL

** What is postgreSQL ?
-> its an advance open-source object-relational database.

** Feature
 - ACID compliant
 - JSON support
 - Strong Indexing
 
 --------------------------------------------------------------
 
# Example

* install
-> sudo apt install postgresql postgresql-contrib

* Service start
-> sudo service postgresql start

-> sudo -i -u postgres
-> psql
-> CREATE DATABASE college;
-> \l
-> \c college
